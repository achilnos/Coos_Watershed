Hello scientists, devs, or engineers,
As part of a collaboration with the Coos Watershed Association I'm working on a hydrologic model for the Coos Bay area. 
This model will use tidal gauges, streamflow gauges, and doppler precipation data as inputs. We use the shallow water lattice boltzmann model of the streams and tides. Rainfall will distributed using using an Upslope area / Muskingum-Cunge method

This repositiory will be used to track progress and curate finished versions of the various componets.
